Its contents are written in HTML, or HyperText Markup Language. HTML documents contain regular text and multimedia contents, structured by "tags" to make those contents readable and meaningful for the browser, scripts, and styles.

For example, the contents of this paragraph are surrounded by a "p" tag.

And so are these. Right click this page (wherever, really) and select "inspect element" to open up Developer tools and see the html that structures this document.

The structural semantics HTML provides for content adds meta-information to it. HTML tags can have Attributes that define additional information about the content, like styles, links, data, identifying information like classes and IDs, etc.
