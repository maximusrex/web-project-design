p5.js

P5.js is a free, open-source JavaScript library for creative coding.

You can use p5.js by calling the .js library in your html page (you can find the CDN link here), and invoking P5's native setup() and draw() functions in javascript.

P5 makes use of the HTML Canvas element. In the setup() function for this "sketch", the function createCanvas(w,h) is called, which creates an HTML Canvas element and injects it into the page.

The draw() loop, predictably, defines what is "drawn on" the canvas. This can be anything from code to generate drawings with shape primitives (like line(), ellipse(), or rect()) to user interaction with mouse events, keycodes, etc. The canvas element in p5 also supports images, audio, 3D, and video.

P5.js also has an online editor where you can experiment and create p5 sketches in the browser.
