Adding React to your HTML site

React is a component-based library for building UI in JavaScript Apps. It uses JSX, an extension of JavaScript, which allows you to declare custom UI components.

You can use React components without building a React app from scratch by calling the CDN files in script tags in your document. Follow this tutorial to practice on your own with this method.

The React Homepage also has a few simple examples with an in-browser editor so you can get used to what react components look like and how they work
