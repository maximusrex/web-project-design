ML5 is a javascript library for approachable machine learning in the browser. These examples work with p5.js and the canvas element.

If you want to learn more, check out Dan Shiffman's extensive Coding Train tutorials on Machine Learning for Beginners.

Also check out the ml5 github examples folder for a ton of other starter code and examples of what you can do with ml5.
