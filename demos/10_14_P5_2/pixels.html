Manipulating Pixels

Using loadPixels() and updatePixels(), you can access and change the values in the pixel array.

Each pixel on the canvas is represented by 4 consecutive values in the array with numbers (0-255) for the red, green, blue, and alpha values (in that order).

Check the code here for a useful algorithm for accessing and changing the values of the pixel array.

See the Coding Train Tutorial on the pixel array for a full demo.
